# 🚛 Telegram бот для управління рейсами

Автоматизована система управління рейсами, звітністю та обліком палива для транспортних компаній.

## 🎯 Основні можливості

### 👥 Ролі користувачів
- **Водії**: створення звітів про рейси, додавання чеків АЗС, примітки
- **ТО**: управління технічним обслуговуванням, заміна масла, норми розходу
- **Бухгалтери**: перегляд звітів, аналіз витрат
- **Адміністратори**: повне управління системою

### 📋 Функціонал для водіїв
- ✅ Звіти про рейси з примітками
- ✅ Завантаження чеків АЗС з OCR розпізнаванням
- ✅ Автоматичний розрахунок пробігу
- ✅ Нагадування о 13:00 за українським часом

### 🔧 Функціонал для ТО
- ✅ Реєстрація заміни масла
- ✅ Ведення журналу ТО
- ✅ Управління нормами розходу палива
- ✅ Нагадування про заміну масла

### 💰 Функціонал для бухгалтерів
- ✅ Звіти по рейсах з примітками
- ✅ Звіти по пальному
- ✅ Звіти по ремонтах
- ✅ Звіти по заміні масла

### 👑 Функціонал для адміністраторів
- ✅ Створення турів з примітками та типами
- ✅ Управління ролями користувачів
- ✅ Редагування довідників
- ✅ **🔔 Ручні нагадування про звіти**
- ✅ **🛢️ Миттєва перевірка заміни масла**
- ✅ Повний доступ до всіх функцій

## 🆕 Нові можливості (версія 2.2.0)

### 📝 Примітки до рейсів
Водії можуть додавати примітки до кожного рейсу для запису:
- Особливостей маршруту
- Проблем в дорозі
- Додаткової інформації

### 🔄 Подвійні тури
Підтримка турів, які виконуються двома водіями:
- **1-ша частина**: перший водій
- **2-га частина**: другий водій

### ⏰ Правильний час нагадувань
- Нагадування про звіти: **13:00-18:00** за українським часом
- Нагадування про заміну масла: **13:00** за українським часом

### 🔒 Обмеження доступу
Норма розходу палива тепер доступна тільки ролі **ТО** та **адміністраторам**

### 🔔 Ручні нагадування для адміністраторів
Адміністратори можуть миттєво надсилати нагадування:
- **Нагадування про звіти**: всім водіям без звітів за сьогодні
- **Перевірка заміни масла**: всім, кому потрібна заміна масла
- **Статистика**: показує результати надсилання
- **Переваги**: не потрібно чекати автоматичного cron

## 🚀 Швидкий старт

### 1. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 2. Налаштування
1. Отримайте токен бота від [@BotFather](https://t.me/BotFather)
2. Створіть Google Sheets API ключ
3. Оновіть конфігурацію в `v1/betav2.py`:
   ```python
   BOT_TOKEN = "ваш_токен_бота"
   GOOGLE_SHEET_NAME = "назва_вашої_таблиці"
   GOOGLE_CREDS_JSON = {...}  # ваші_google_credentials
   ```

### 3. Запуск
```bash
cd v1
python betav2.py
```

### 4. Тестування
```bash
python test_bot.py
```

### Аркуш "Водії"
| user_id | first_name | last_name | phone_number | fuel_card | vehicles_json | role |

### Аркуш "Рейси" 
| ПІБ Водія | Час звіту | Дата звіту | Номер авто | Марка авто | Тип палива | Норма розходу | Номер туру | Табель | Паливна картка | КМ | Фактична заправка | **Примітка** |

| Номер туру | Відстань км | Ким створено | Дата створення | **Примітка** | **Тип туру** |

| Дата | Номер авто | Виконавець (ТО) | Ціна, грн | Кількість, л |

## 🔧 Технічні деталі

### Використані технології
- **Python 3.11+**
- **aiogram 3.0** - Telegram Bot API
- **gspread** - Google Sheets API
- **easyocr** - розпізнавання тексту з чеків
- **aiocron** - планувальник завдань

### Архітектура
- **FSM (Finite State Machine)** для управління станами діалогу
- **Асинхронне програмування** для високої продуктивності
- **Кешування** для оптимізації роботи з Google Sheets
- **OCR** для автоматичного розпізнавання чеків

### Безпека
- Перевірка ролей користувачів
- Валідація введених даних
- Логування всіх операцій
- Обмеження доступу до критичних функцій

## 📱 Команди бота

### Загальні команди
- `/start` - реєстрація/вхід в систему

### Для водіїв
- **📋 Надіслати звіт про рейс** - створення звіту з примітками
- **⛽️ Надіслати чек АЗС** - завантаження чека з OCR

### Для ТО
- **🔧 Панель ТО** - доступ до функцій ТО

- **🛠️ Тікети з поломками** - управління поломками

### Для адміністраторів
- **👑 Панель адміністратора** - повне управління
- **🗺️ Створити новий тур** - створення турів з примітками
- **👥 Управління ролями** - зміна ролей користувачів
- **🔔 Нагадування про звіти** - ручне надсилання нагадувань водіям
- **🛢️ Перевірка заміни масла** - миттєва перевірка необхідності заміни масла

## 🐛 Відомі обмеження

1. **NumPy версія**: Попередження про сумісність NumPy 1.x/2.x (не критично)
2. **OCR точність**: Залежить від якості фото чека
3. **Google Sheets API**: Обмеження на кількість запитів
